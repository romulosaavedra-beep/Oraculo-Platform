Gemini, precisamos criar o endpoint para deletar documentos. Modifique o arquivo `backend/app/api/v1/endpoints/workspace.py`. Adicione uma nova rota `DELETE` no caminho `/documents/{document_id}`. Esta rota deve ser protegida com `Depends(get_current_user)`. Dentro da função, ela deve:
1.  Receber o `document_id` da URL.
2.  Verificar se o documento pertence ao usuário logado.
3.  Se pertencer, usar o cliente Supabase para deletar o registro da tabela `documents`.
4.  Retornar uma mensagem de sucesso.
Lembre-se que o nosso gatilho no Supabase cuidará de deletar o arquivo do Storage e os chunks associados.
